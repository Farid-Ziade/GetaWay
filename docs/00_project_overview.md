# 00 — Project Overview

## What is GetaWay?

GetaWay is an AI-powered weekend getaway planner. Users sign in, share their location, set a budget, and receive a personalized 2-day trip plan generated by an AI that considers the weather, nearby places, and their past trips.

## Core Features

| Feature | Status |
|---|---|
| Landing page | Phase 2 |
| Email + Google Sign-In | Phase 4 |
| Location permission + map | Phases 6–7 |
| Nearby places | Phase 8 |
| Weather-aware planning | Phase 9 |
| AI trip generation | Phase 11 |
| Budget adjustment | Phase 12 |
| Saved trips (no repeats) | Phase 13 |

## Tech Stack

| Layer | Technology | Reason |
|---|---|---|
| Frontend | React + Vite | Fast SPA, no SSR needed for logged-in app |
| Routing | React Router v7 | Standard SPA routing |
| Auth | Firebase Authentication | Existing project, Google + email |
| Database | Firebase Firestore | Per-user saved trips with security rules |
| Maps | Google Maps JS API via @react-google-maps/api | Official SDK |
| Weather | OpenWeatherMap API | Free, reliable |
| Backend | Node.js + Express | Keeps OpenAI key server-side |
| AI | OpenAI (gpt-4o-mini) | Backend only — never exposed to frontend |
| Hosting | Firebase Hosting + backend server | Co-deployable |
